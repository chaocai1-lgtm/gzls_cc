## 🔒 Streamlit Cloud 部署步骤

### 1. 配置 Secrets

在 Streamlit Cloud 应用设置中，添加以下密钥：

```toml
# 复制 .streamlit/secrets.toml 的内容
# 或使用真实的生产环境密钥

NEO4J_URI = "bolt://your-host:port"
NEO4J_USERNAME = "your-username"
NEO4J_PASSWORD = "your-password"

ELASTICSEARCH_CLOUD_ID = "your-cloud-id"
ELASTICSEARCH_USERNAME = "your-username"
ELASTICSEARCH_PASSWORD = "your-password"

DEEPSEEK_API_KEY = "sk-your-api-key"
```

### 2. 部署应用

1. 访问 [share.streamlit.io](https://share.streamlit.io)
2. 连接 GitHub 仓库
3. 选择分支：`main`
4. 主文件路径：`高分子自适应学习系统/app.py`
5. 保存并部署

## ⚠️ 重要提醒

1. **本地 secrets.toml 文件**包含真实密钥，请妥善保管，切勿分享
2. **GitHub 仓库建议设为私有**，避免代码泄露
3. **定期更换密钥**，尤其是怀疑泄露时
4. **部署后测试**所有功能是否正常工作

## 📞 问题排查

如果部署后遇到问题：

1. **数据库连接失败**
   - 检查 Streamlit Secrets 配置是否正确
   - 确认 Neo4j 允许外网访问
   - 验证用户名和密码

2. **案例检索不工作**
   - 检查 Elasticsearch 配置
   - 确认索引已创建

3. **AI 功能无响应**
   - 验证 DeepSeek API Key 有效性
   - 检查 API 额度是否充足
